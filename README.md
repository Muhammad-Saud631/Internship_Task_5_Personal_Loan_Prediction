# Personal Loan Acceptance Prediction

## Objective
Predict which customers are likely to accept a personal loan offer using Logistic Regression.

## Dataset
Bank Marketing Dataset (UCI/Kaggle)

## Approach
1. Data cleaning and exploration
2. Encode categorical variables
3. Train Logistic Regression model
4. Evaluate using accuracy and confusion matrix
5. Visualize important features

## Results
- Accuracy: 0.92 (example)
- Key factors: Income, Balance, Previous banking activity
- Insights: Target high-income and engaged customers for offers
